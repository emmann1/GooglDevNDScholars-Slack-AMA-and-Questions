About the project

This project began from the inspiration of Doha Helmy's NDTranscripts, with my personal ideas and minimal design. I wanted to build this with no back-end required and no SQL at all.

So I used just Javascript and JSON files to grab questions from the file and display them. Filters are also added to chose betwen sessions of different dates and subject of discussion. With the checkboxes you can choose betwen tracks, Android Nanodegree(AND), Front End Nanodegree(FEND) and Mobile Web Specialist Nanodegree(MWS).

The search bar searches through all questions and answear by matching firstly the entire searched query. If nothing is found it then separates the words and searches separately by matching every word in one question. For example: if you search by the term "google sites" it tries to match the entire term in one question, if it doesn't return anything, then it separates "google" and "sites" and tries to find an item having both of the words. It also mathches to entire words, searching incomplete words wont work(like inputing "udaci" and expecting to find "udacity").

I found this search tactic most useful because it matches the entire word not just parts of it, and instead of searching for seaparate words and having hundreds of results, it returns every instance with all searched terms.
